#  Simple Springboot + EmbebbedMongoDB(memory database)


### To run
mvn package spring-boot:run

### To test

mvn test

### Endpoints documentation

* [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)



### MongoDB Embebbed Config.
* Database: test
* Collections: product
* url: localhost
* port: 27018
